# To Do List React Redux

Este proyecto fue creado con [Create React App](https://github.com/facebookincubator/create-react-app).

```
git clone https://github.com/PaulaPonce/todo-list-react.git
cd todo-list-react
npm install
npm start

http://localhost:3000/
```
